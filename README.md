# DrugWars.io

The drugs are virtual, but the money is real

## Installation

Clone the repo & install dependencies.

```bash
git clone https://github.com/FutureShockco/drugwars.io.git
cd drugwars.io
npm install 
```
Start the client 
```bash
npm run serve
```
Visit http://localhost:8080
